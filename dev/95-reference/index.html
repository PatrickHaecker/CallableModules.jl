CallableModules.@module_main — Macro
@module_main funcdef

Make the enclosing module callable by forwarding MyModule(args...; kwargs...) to the annotated function.

Examples

module MyApp
    using CallableModules
    @module_main function main(args...; kwargs...)
        println("args=$args, kwargs=$kwargs")
    end
end

MyApp(1, 2; verbose=true)
# args=(1, 2), kwargs=Base.Pairs(:verbose => true)

Both long-form (function f(...) ... end) and short-form (f(x) = ...) definitions are supported.
